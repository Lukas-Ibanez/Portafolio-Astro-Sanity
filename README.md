# Portafolio Astro Sanity

Portafolio profesional de Lukas Ibáñez construido con Astro, Tailwind CSS, GSAP y Sanity.

Regla central del proyecto: cero requests externos en runtime. Sanity y GitHub se resuelven en build-time.

## Comandos

```bash
npm run dev
npm run build
npm run preview
```

Astro 6 requiere Node `>=22.12.0`. En esta máquina los scripts usan un Node portable local en `.local-node/`, ignorado por git.
